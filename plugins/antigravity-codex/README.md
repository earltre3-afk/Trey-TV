# Antigravity Codex

This plugin is a starter scaffold for using Codex from Antigravity-oriented workflows.

## Contents

- `.codex-plugin/plugin.json` defines the plugin manifest.
- `skills/antigravity-codex/SKILL.md` provides the first Codex-facing workflow.
- `.mcp.json` is reserved for a future Antigravity MCP bridge.
- `.app.json` is reserved for a future app connector surface.
- `scripts/` is reserved for prompt/context bridge helpers.
