# Trey TV Luxury Logo Card Deck

Generated full 52-card PNG deck for Trey TV Games / PixiJS.

Install by copying the `public/` folder into the Trey TV app root.

Expected runtime path:
`public/assets/games/cards/trey-tv-luxury/`

Includes:
- 52 face cards
- `card-back.png` using the official Trey TV logo
- folder names expected by the loader: `blades`, `soul`, `flame`, `keys`

Suit mapping:
- Blades -> Spades -> S
- Soul -> Hearts -> H
- Flame -> Diamonds -> D
- Keys -> Clubs -> C

Card size: 420x588px PNG.
